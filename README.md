        ```bash

        ```

    2. Configure the project:
        ```bash
        cmake ..
        ```

    3. Build the project:
        ```bash
        cmake --build .
        ```

    **Option 2: Premake**

    1. From the root directory, generate build files:
        ```bash
        premake5 gmake2
        ```

    2. Build the project:
        ```bash
        make
        ```

    ### Running

    From the root directory, run:
    ```bash
    ./bin/Debug/Sandbox
