
# Void

A game engine built with C++, CMake, and spdlog.

## Getting Started

### Build Instructions

1. Create a build directory from the root:
    ```bash
    mkdir build
    cd build
    ```

2. Configure the project:
    ```bash
    cmake ..
    ```

3. Build the project:
    ```bash
    cmake --build .
    ```

### Running

From the root directory, run:
```bash
./bin/Debug/Sandbox
```

## Dependencies

- C++
- CMake
- spdlog
